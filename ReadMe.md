## Сортировка слиянием
Сортировка слиянием представляет собой алгоритм «разделяй и властвуй». Он делит входной массив на две половины, вызывает себя для этих двух половин, а затем объединяет две отсортированные половины. Функция merge() используется для слияния двух половин. Слияние (arr, l, m, r) — это ключевой процесс, который предполагает, что arr[l..m] и arr[m+1..r] отсортированы, и объединяет два отсортированных подмассива в один.

##Сложность алгоритма
У сортировки слиянием постоянная сложность она
всегда равна (n*log(n))
Также сортировка требует дополнительной памяти по размеру исходного массива.
